Quantitative Momentum Strategy (VCE Project)

Introduction
This is a quantitative trading project developed by a Year 9 student in Melbourne.
The goal is to analyze stock market trends using Python and statistical methods, focusing on the Momentum Strategy.

Tech Stack
* **Language:** Python 3.10
* **Libraries:** Pandas, Matplotlib, NumPy
* **Tools:** Jupyter Notebook

Key Features
1.  **Data Analysis:** Calculates daily returns and profit per share.
2.  **Strategy Logic:** Identifies "Buy" signals based on price momentum (>1% daily gain).
3.  **Visualization:** Generates bar charts to compare stock performance.

Future Goals
* Connect to Yahoo Finance API for real-time data.
* Implement Machine Learning (Linear Regression) for price prediction.
* Backtest the strategy against historical data (2020-2025).
